# Acme Shop - E-Commerce Case Study

E-commerce application used as a case study for progressive migration
to Hexagonal Architecture with [HexaGlue](https://hexaglue.io).

## Structure

```
├── legacy/       # Original monolithic application
├── hexagonal/    # Migrated application (hexagonal architecture + HexaGlue)
```

## Quick Start

### Legacy Application

```bash
cd legacy
mvn clean compile
```

### Hexagonal Application

```bash
cd hexagonal
mvn clean verify
```

Run the application:

```bash
cd hexagonal
mvn spring-boot:run
```

Then open [http://localhost:8080/swagger-ui.html](http://localhost:8080/swagger-ui.html).

## Migration Summary

| Aspect | Legacy | Hexagonal |
|--------|--------|-----------|
| Architecture | Monolithic layers | Hexagonal (ports & adapters) |
| Domain model | Anemic (JPA entities) | Rich domain model |
| Value objects | Primitives | Typed IDs, Money, Address, Quantity |
| Infrastructure | Mixed with domain | Generated by HexaGlue |
| Structure | Single module | Single module (hexagonal packages) |

See [MIGRATION.md](MIGRATION.md) for a detailed migration guide.
